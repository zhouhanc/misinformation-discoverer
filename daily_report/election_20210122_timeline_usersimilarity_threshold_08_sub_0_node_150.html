    1. Domain name
    2. Fakeness score (the higher, the more similar a domain looks to a known fake news sites)
    3. WHOIS information (registrar, organization)
    4. IP address associated with the domain name
  3. IP detailed statistics
    1. location (city, country, region)
    2. hostname
    3. organization
  4. Domain-domain interaction network
    1. Each node is a domain
    2. Two nodes are connected if users who share those two domains have high overlap
    3. Colors represent different communities (based on unsupervised Louvain clustering)
